# CLI Output Examples

This folder stores runnable command examples and captured stdout from `h265nal-cli`.

- `cli-output-examples.md`: command snippets and sample output excerpts.
- `akiyo_qp30_dump.c.out`: full C-format dump output for `video/akiyo.x265.qp_30.265`.
- `akiyo_qp30.c.out`: count-style C output for the same sample.
- `iphone_pretty.out`: pretty/count summary output for `video/iphone_11s.messenger.704x1280.265`.
- `downloads_center.pretty.out`: pretty/count summary output for the Downloads sample.
- `downloads_center.c.out`: count-style C output for the Downloads sample.

These files are generated locally for documentation and troubleshooting.
